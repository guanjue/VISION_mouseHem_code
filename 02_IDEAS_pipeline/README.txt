### The IDEAS scripts can be downloaded from:
https://github.com/seqcode/IDEAS

The user needs to change the directory in the following files and then run IDEAS.
run_IDEAS.sh
run_IDEAS.parafile
run_IDEAS.input
run_IDEAS_lesshetero.sh
run_IDEAS_lesshetero.parafile
run_IDEAS_lesshetero.input

### run IDEAS
bash run_IDEAS.sh

### rerun IDEAS after removing heterogeneous states
bash run_IDEAS_lesshetero.sh
